Požadavek: Robot R4
  Robot R4 je kolaborativní robot Kuka IIWA. Má za úkol finální kompletaci autíčka. Postupně mu jsou na jeho stanici XX přivezeny zkompletované součásti autíčka. Robotu je nejdříve přivezono chassi autíčka. Následuje body (korba) a jako poslední je přivezena kabina.
Kontext:
  Pokud linkaOn == 1

Scénář: Normální běh linky - Robot R4 - brání chassi z vozíčku
  Když shuttleXyAtStationYZ == 1 && edge(stationLocked, 1)
  Pak   robotR4ProgramNumber == 1 && edge(robotR4ProgamStart, 1)

Scénář: Normální běh linky - Robot R4 - pokládání chassi před operátora
  Když  robotR4ProgramNumber == 1 && edge(robotR4ProgamEnd, 1)
  Pak   robotR4ProgramNumber == 2 && edge(robotR4ProgamStart, 1)

Scénář: Normální běh linky - Robot R4 - brání body z vozíčku
  Když  robotR4ProgramNumber == 2 && edge(robotR4ProgamEnd, 1)
  Pak   robotR4ProgramNumber == 3 && edge(robotR4ProgamStart, 1)

Scénář: Normální běh linky - Robot R4 - pokládání body před operátora
  Když  robotR4ProgramNumber == 3 && edge(robotR4ProgamEnd, 1)
  Pak   robotR4ProgramNumber == 4 && edge(robotR4ProgamStart, 1)

Scénář: Normální běh linky - Robot R4 - brání cabin z vozíčku
  Když  robotR4ProgramNumber == 4 && edge(robotR4ProgamEnd, 1)
  Pak   robotR4ProgramNumber == 5 && edge(robotR4ProgamStart, 1)

Scénář: Normální běh linky - Robot R4 - pokládání cabin před operátora
  Když  robotR4ProgramNumber == 5 && edge(robotR4ProgamEnd, 1)
  Pak   robotR4ProgramNumber == 6 && edge(robotR4ProgamStart, 1)
