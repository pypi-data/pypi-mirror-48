# language: cs
Feature: robot R4
  Robot R4 je kolaborativní robot Kuka IIWA. Má za úkol finální kompletaci autíčka. Postupně mu jsou na jeho stanici XX přivezeny zkompletované součásti autíčka. Robotu je nejdříve přivezono chassi autíčka. Následuje body (korba) a jako poslední je přivezena kabina.
Background:
  Pokud linka být zapnutý
Scenario: Pokud je sklad prázdný, když je vozíček XX ve stanici XY, Pak robot vyloží všechny kostičky 3x2.
Given sklad je prázdný
When vozíček XX je ve stanici XY
Then robot vyloží kostičky 3x2
Scenario: Když robot skončí vykládání kostiček 3x2  Pak robot vyloží všechny kostičky 4x2.
When robot skončí vyložit kostiček 3x2
Then robot vyloží kostičky 4x2
Scenario: Když robot dokončí vykládání kostiček 4x2, Pak robot vyloží všechny kostičky 6x2.
When robot dokončí vyložit kostiček 4x2
Then robot vyloží kostičky 6x2
