
Požadavek: Robot R2
  Robot R2 má na starosti kompletaci skládání Body. Body je návstavba za kabinou autíčka. Robot postupně zbírá kostičky potřebené ke kompletaci.
Kontext:
  Pokud linkaOn == 1
  A storeR2Count > 0

Scénář: Normální běh linky - Robot R2 - Skládání Body - braní kostičky číslo 1
  Když shuttleXyAtStationYZ == 1 && edge(stationLocked, 1)
  Pak   robotR2ProgramNumber == 1 && edge(robotR2ProgamStart, 1)

Scénář: Normální běh linky - Robot R2 - Skládání Body - pokládání kostičky číslo 1
  Když  robotR2ProgramNumber == 1 && edge(robotR2ProgamEnd, 1)
  Pak   robotR2ProgramNumber == 2 && edge(robotR2ProgamStart, 1)

Scénář: Normální běh linky - Robot R2 - Skládání Body - braní kostičky číslo 2
  Když  robotR2ProgramNumber == 2 && edge(robotR2ProgamEnd, 1)
  Pak   robotR2ProgramNumber == 3 && edge(robotR2ProgamStart, 1)
