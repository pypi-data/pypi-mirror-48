# language: cs

Požadavek: robot R1
      Robot R1 má na starosti kompletaci skládání Chassi. Robot postupně sbírá kostičky potřebné ke kompletaci.
Kontext:
  Pokud lineOn

Scénář: Pokud je sklad R1 prázdný a když je vozíček XX ve stanici XY, pak robot R1 začne vykládat všechny kostičky 3x2.
  Pokud storageR1Count == 0
  Když shuttleXXAtStationXY
  Pak robotR1ProgramNumber == 4 && robotR1ProgramStarted

Scénář: Když robot R1 skončil vykládání kostiček 3x2, pak robot začne vykládat všechny kostičky 4x2.
  Když robotR1ProgramNumber == 4 && robotR1ProgramEnded
  Pak robotR1ProgramNumber == 5 && robotR1ProgramStarted

Scénář: Když robot R1 skončil vykládání kostiček 4x2, pak robot R1 začne vykládat všechny kostičky 6x2.
  Když robotR1ProgramNumber == 5 && robotR1ProgramEnded
  Pak robotR1ProgramNumber == 6 && robotR1ProgramStarted
