Na úvod můžeme zadefinovat, co znamenají některé slovní obraty, např. - 

"Robot sebere kostičku #" v kroku "Když" znamená, že se spustí program "sebratKosticku#" v daném robotu. Naproti tomu tentýž výrok v kroku "Pak" znamená, že skončil program "sebratKostičku#".

Čísla programů lze přiřadit při analýze sekvence úkonů robotů tak, aby byla unikátní pro unikátní operaci.
